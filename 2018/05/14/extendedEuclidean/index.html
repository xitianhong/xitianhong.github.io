扩展欧几里得算法

1、定理：对于非零整数a和b，存在整数x和y，使得ax + by = (a,b)

2、gcd: 最大公约数 a 和 b 的最大公约数写作( a , b )

3、b|a: 表示b能整除 a , 可写成 a = kb;

4、线性组合就是形如a1 k1 + a2 k2 + a3 * k3…这样的式子

求证：sa + tb = (a, b).

假设 d 是 a 和 b 的线性组合的最小正整数结果: 写作: d = ax + by;

首先证明 d | a (a可以被d整除):

通过带余除法, a 除以 b 可以写成 a = qd + r (0 <= r < d);
所以 r = a − dq 
把d = ax + by; 带入上r = a − dq 得到:
r = a − q(sa + tb) 
r = a − qsa − qtb 
r = (1 − qs)a + (−qt)b
所以r也是 a 和 b 的线性组合;
但是由于 d 是 a 和 b 的线性组合的最小正值并且0 <= r < d
所以 r 只能是0;
所以 a = qd + r = qd
所以 a 是 b 的整数倍,b是a的约数;
所以 d | a;
同样的方法也可以推出 d | b.
所以 d 是 a 和 b 的公约数;

现在证明 d 是 a 和 b 的最大公约数:

假设 p 为 a 和 b 的最大公约数，写作(a, b) = p;所以 p > d;
因为 p | a 可以表示成 a = mp, p | b 可以表示成b = np, 并且d = ax + by
所以 d = xmp + ynp = (xm + yn)*p;
所以 p | d;
所以 d >= p;
但是 p > d，与条件矛盾;
所以 不存在比 d 更大的公约数；
所以 d 就是 a 和 b 的最大公约数;
所以存在 d = ax + by = (a, b)
